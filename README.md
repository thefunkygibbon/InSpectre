# 🛡️ InSpectre

A Dockerized network security suite focused on visibility, history, and active defense.

## Features
- **Live Discovery**: Real-time ARP scanning to track who is on the network.
- **Historical Logs**: Persistent storage of every device that has ever connected.
- **Security Audit**: Integrated Nmap vulnerability scanning.
- **The "Kill-Switch"**: ARP Spoofing to pause internet access for specific devices.

## Tech Stack
- **Backend**: Python (FastAPI)
- **Database**: PostgreSQL
- **Scanning**: Scapy + Nmap
- **Frontend**: React + Tailwind (Coming Soon)
